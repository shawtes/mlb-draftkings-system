📋 HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

For CASH GAMES (50/50, H2H):
  → Select "QB + WR" or "No Stack"
  → Safe and reliable

For GPP TOURNAMENTS:
  → Select "QB + 2 WR" or "QB + WR + TE"
  → More aggressive, higher ceiling

For SHOOTOUT GAMES:
  → Select "Game Stack"
  → Bet on high-scoring back-and-forth

For HEDGE PLAYS:
  → Select "Bring-Back"
  → QB+WR + opponent RB protects downside
